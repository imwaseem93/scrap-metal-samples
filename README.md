## üöÄ Quick Deploy to GitHub Pages

### Step 1: Create a New Repository
1. Go to [github.com/new](https://github.com/new)
2. Name it: `scrap-metal-samples` (or any name you prefer)
3. Set to **Public**
4. Click **Create repository**

### Step 2: Upload Files
1. Click **"uploading an existing file"** link
2. Drag and drop ALL files from this folder:
   - `index.html`
   - `sample-1-metalworks.html`
   - `sample-2-alloytrade.html`
   - `sample-3-metalvault.html`
3. Click **Commit changes**

### Step 3: Enable GitHub Pages
1. Go to repository **Settings**
2. Scroll to **Pages** (left sidebar)
3. Under "Source", select **Deploy from a branch**
4. Select **main** branch and **/ (root)** folder
5. Click **Save**

### Step 4: Get Your Live URL
Wait 1-2 minutes, then your site will be live at:
```
https://YOUR-USERNAME.github.io/scrap-metal-samples/
```
